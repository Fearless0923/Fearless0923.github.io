🍭PHP 运算符

🍭算术运算符:

x + y x - y x * y x / y x % y - x a . b
加 减 乘 除 模（除法的余数） 取反 并置

整除运算符 intdiv()，该函数返回值为第一个参数除于第二个参数的值并取整（向下取整）。

<?php
var_dump(intdiv(16, 5));
?>

结果为3

🍭组合比较符(PHP7+):

组合比较符（combined comparison operator）也称为太空船操作符，符号为 <=>。组合比较运算符可以轻松实现两个变量的比较，也不仅限于数值类数据的比较。

$c = $a <=> $b;
  • 如果 $a > $b, 则 $c 的值为 1。
  • 如果 $a == $b, 则 $c 的值为 0。
  • 如果 $a < $b, 则 $c 的值为 -1。

🍭运算符优先级:

运算符的优先级口诀：括（号）、单（操作数）、算（术）、移（位）、关（系）；位（运算符）、逻（辑）、条（件）、赋（值）、逗（号）。or 和 ||，&& 和 and 都是逻辑运算符，效果一样，但是其优先级却不一样。我们通过括号的配对来明确标明运算顺序，而非靠运算符优先级和结合性来决定，通常能够增加代码的可读性。

🍭PHP语句

🍭if 语句：

if 语句用于仅当指定条件成立时执行代码。

🍭PHP - if…else 语句:

在条件成立时执行一块代码，条件不成立时执行另一块代码。

🍭PHP - if…elseif…else 语句:

在若干条件之一成立时执行一个代码块。

🍭Switch 语句：

switch 语句用于根据多个不同条件执行不同动作，有选择地执行若干代码块之一。

工作原理：首先对一个简单的表达式 n（通常是变量）进行一次计算。将表达式的值与结构中每个 case 的值进行比较。如果存在匹配，则执行与 case 关联的代码。代码执行后，使用 break 来阻止代码跳入下一个 case 中继续执行。default 语句用于不存在匹配（即没有 case 为真）时执行。

🍭PHP 数组array()

数组是一个能在单个变量中存储多个值的特殊变量。

  • 数值数组 - 带有数字 ID 键的数组
  • 关联数组 - 带有指定的键的数组，每个键关联一个值
  • 多维数组 - 包含一个或多个数组的数组

🍭count() 函数:

count() 函数用于返回数组的长度（元素的数量)。

🍭PHP 数值数组：

这里有两种创建数值数组的方法：

自动分配 ID 键（ID 键总是从 0 开始）

$cars=array("rose","lisa","jennie""jisoo");

人工分配 ID 键：

$cars[0]="rose";
$cars[1]="lisa";
$cars[2]="jennie";
$cars[3]="jisoo";

🍭遍历数值数组:

遍历并打印数值数组中的所有值，可以使用 for 循环。

🍭关联数组:

使用分配给数组的指定的键的数组。

法一:

$age=array("Jane"=>"35","Ben"=>"37","Joe"=>"43");

法二：

$age['Jane']="35";
$age['Ben']="37";
$age['Joe']="43";

🍭PHP 数组排序

数组中的元素可以按字母或数字顺序进行降序或升序排列。

  • sort() - 对数组进行升序排列
  • rsort() - 对数组进行降序排列
  • asort() - 根据数组的值，对数组进行升序排列
  • ksort() - 根据数组的键，对数组进行升序排列

🍭PHP 超级全局变量

超级变量在全部作用域中都可用，不需要特别说明。

🍭$GLOBALS

包含了全部变量的全局组合数组

🍭$_SERVER

包含了头信息(header)、路径(path)、以及脚本位置(script locations)等信息的数组，数组中的项目由 Web 服务器创建。

🍭$_REQUEST

用于收集HTML表单提交的数据。以下实例显示了一个输入字段（input）及提交按钮(submit)的表单(form)。 当用户通过点击 “Submit” 按钮提交表单数据时, 表单数据将发送至

标签中 action 属性中指定的脚本文件。 在这个实例中，我们指定文件来处理表单数据。如果你希望其他的PHP文件来处理该数据，你可以修改该指定的脚本文件名。 然后，我们可以使用超级全局变量 $_REQUEST 来收集表单中的 input 字段数据。

<html>
<body>
 
<form method="post" action="<?php echo $_SERVER['PHP_SELF'];?>">
Name: <input type="text" name="fname">
<input type="submit">
</form>
 
<?php 
$name = $_REQUEST['fname']; 
echo $name; 
?>
 
</body>
</html>

🍭$_POST

用于收集表单数据，在HTML form标签的指定该属性："method=“post”。(用法与上述$_REQUEST类似)

🍭$_GET

$GET 同样被广泛应用于收集表单数据，在HTML form标签的指定该属性："method=“get”。$_GET 也可以收集URL中发送的数据。

🍭PHP 循环

🍭while

只要指定的条件成立，则循环执行代码块。

语法如下：

while (条件)
{
    要执行的代码;
}

以下实例变量 i 的初始值为 9。然后只要 i 小于或者等于 13，while 循环将继续运行。循环每运行一次，i 就会递增 1：

<html>
<body>

<?php
$i=9;
while($i<=13)
{
    echo "The number is " . $i . "<br>";
    $i++;
}
?>

</body>
</html>

运行结果如下：

The number is 9
The number is 10
The number is 11
The number is 12
The number is 13

🍭do…while 语句

do…while 语句执行一次代码，然后检查条件，只要条件成立，就会重复进行循环。

语法如下:

do
{
    要执行的代码;
}
while (条件);

🍭for 循环

for 循环用于预先知道脚本需要运行的次数的情况。

语法如下：其中的初始值和增量参数可为空或者有多个表达式（用逗号分隔）。

for (初始值; 条件; 增量)
{
    要执行的代码;
}

实例如下:定义一个初始值为 i=8的循环，只要变量 i 小于或者等于 10，循环将继续运行。循环每运行一次，变量 i 就会递增 1：

<?php
for ($i=8; $i<=10; $i++)
{
    echo "i= " . $i . PHP_EOL;
}
?>

结果如下:

i= 8
i= 9
i= 10

🍭foreach 循环

foreach 循环用于遍历数组。

语法如下：

foreach ($array as $value)
{
    要执行代码;
}

每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。

foreach ($array as $key => $value)
{
    要执行代码;
}

语法与上面类似，但下面的语法中元素的键与值都会被赋值给 $key 和 $value 变量。

🍭PHP 函数

函数是通过调用函数来执行的,可以在页面的任何位置调用函数。

🍭创建 PHP 函数

<?php
function functionName()
{
    // 要执行的代码
}
?>
  • 函数的名称应该提示出它的功能
  • 函数名称以字母或下划线开头（不能以数字开头）
<?php
function write_song()
{
    echo "wonderful u";
}
 
echo "I want to listen to ";
write_song();
?>

结果为 ：I want to listen to wonderful u.

🍭PHP 函数 - 添加参数

参数类似变量,参数就在函数名称后面的一个括号内指定。

<?php
function idol($combination,$punctuation)
{
    echo $combination . " blackpink" . $punctuation . "<br>";
}
 
echo "My first idol is ";
idol("rose",".");
echo "My second idol is ";
idol("lisa","!");
echo "My third idol is ";
idol("jennie","?");
echo "My forth idol is ";
idol("jisoo","?");
?>

输出如下：

My first idol is rose blackpink.
My second idol is lisa blackpink!
My third idol is jennie blackpink?
My forth idol is jisoo blackpink?

🍭PHP 魔术常量

魔术常量的值随着它们在代码中的位置改变而改变。魔术常量的两边都要加上两个下划线。

🍭LINE

文件中的当前行号。

<?php
echo '这是第 " '  . __LINE__ . ' " 行';
?>

以上实例输出结果为：

这是第 “ 2 ” 行

🍭FILE

文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名。

🍭DIR

文件所在的目录。如果用在被包括文件中，则返回被包括的文件所在的目录。

它等价于 FILE，如果是根目录，则目录中名包括末尾的斜杠。

🍭FUNCTION

返回该函数被定义时的名字（区分大小写）。

🍭CLASS

返回该类被定义时的名字（区分大小写），类名包括其被声明的作用区域。

🍭METHOD

返回该方法被定义时的名字（区分大小写）。

🍭NAMESPACE

当前命名空间的名称（区分大小写），此常量是在编译时定义的。
